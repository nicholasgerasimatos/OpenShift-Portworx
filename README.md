# OpenShift-Portworx

This project integrates Portworx storage with OpenShift clusters, offering a robust and scalable storage solution for containerized applications.

## Features

- **Dynamic Provisioning:** Automatically manage storage volumes.
- **High Availability:** Ensures persistent data with redundancy.
- **Scalability:** Easily scale storage with cluster growth.
- **Security:** Secure data handling and access.

## Prerequisites

- OpenShift Container Platform
- Portworx compatible cluster
- Proper network and storage configuration
